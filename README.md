# I2C-LED-Driver

#### Tested on:
* Arduino Uno
* Arduino Teensy

#### Working:
* Up to 500 LEDs (300 Tested)
* Solid color
* Knight rider single color
* Knight rider rainbow
* Full strip rainbow
